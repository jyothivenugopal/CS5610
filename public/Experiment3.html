Demonstrating AngularJS Routing Capabilities

Purpose of the Experiment

This experiment is used to demonstrate the ng-route capabilities of AngularJS. It works in the following way,

When one of the links is clicked, the corresponding template and controller in app.js is called, which in turn calls the respeoctive html page.

Demo of the experiment

Click to view page1 Click to view page2

Code Snippets

myApp.config(['$routeProvider',
  function($routeProvider) {
    $routeProvider.
      when('/pg1', {
	templateUrl: 'page1.html',
	controller: 'pg1ctrl'
      }).
      when('/pg2', {
	templateUrl: 'page2.html',
	controller: 'pg2ctrl'
      }).
      otherwise({
	redirectTo: '/pg1'
      });
}]);
